README for Part 1 Homework (Bezier Curve)

Run (open) index.html in any web browser (preferably Google Chrome) or use the live demo link. No downloads or compiling necessary.

Source code for the bezier curve is in bezier.js.

Libraries used include bootstrap for styling. All shapes are drawn with built in HTML Canvas methods.

Live demo link: https://nartexyu.github.io/Bezier-Curve/
